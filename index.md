# Welcome to DataHarness

DataHarness is a unified abstraction layer that brings together multiple table sources (Kafka, databases, lakehouses) into a single abstract table interface.

## Quick Links

- **[Documentation](/DataHarness/docs/)** - Learn how to use DataHarness
- **[Blog](/DataHarness/blog/)** - Read articles about DataHarness

## Features

- **Multi-Source Unification**: Seamlessly integrate data from Kafka, CockroachDB, Iceberg, and more
- **Schema Management**: Centralized schema tracking and management
- **Time-Based Queries**: Read data as of specific timestamps or offsets
- **gRPC Interface**: Efficient client-server communication
